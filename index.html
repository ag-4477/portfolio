Research

  • ニューラルレンダリング（NeRF）を用いた屋根上積雪状態の定量化に関する研究
    ドローン空撮動画からNeRFを用いて家屋の3次元メッシュモデルを生成し、Blender上で積雪部分のみを抽出して雪厚や雪庇（せり出し幅）を自動で定量化する手法を提案。簡易モデルにおいて定量化を実現。（国内学会発表）
    📄 GitHub Repository
  • 複数のYOLO分類モデルを用いた農作物病害のリアルタイム識別に関する比較検証
    人手不足に直面する水田での作物管理自動化を目的に、約6,600枚の画像を用いてYOLO各バージョン（v5, v8, v11）の性能を比較。YOLOv8mで精度99.9%を達成し、かつYOLOv11nが約1/10の計算量で同等の高精度（99.8%）を示すことを実証。（国際学会発表・査読あり）
